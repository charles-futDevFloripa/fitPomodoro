Funções Relacionadas ao funcionamento do Pomodoro

startPomodoro()
O QUE FAZ:
A função startPomodoro() é responsável por iniciar o temporizador do Pomodoro, controlando a contagem regressiva baseada na duração configurada pelo usuário. Se o Pomodoro não estiver atualmente rodando, esta função prepara e inicia a contagem, atualiza a interface do usuário para refletir o tempo restante, e, ao término, notifica o usuário através de um som, além de exibir o modal de alongamento.
A FUNÇÃO:
function startPomodoro() {
            const playPauseIcon = document.getElementById('playPauseIcon');
            if (!isPomodoroRunning) {
              if (!countdownInterval) {
                if (timeLeftInSeconds <= 0) {
                  const durationMinutes =
                    parseInt(localStorage.getItem('pomodoroDuration')) || 25;
                  timeLeftInSeconds = durationMinutes * 60;
                }
          
                updateCountdownDisplay(); 
          
                countdownInterval = setInterval(() => {
                  if (timeLeftInSeconds > 0) {
                    timeLeftInSeconds--;
                    updateCountdownDisplay();
                  } else {
                    clearInterval(countdownInterval);
                    countdownInterval = null; 
                    isPomodoroRunning = false;
                    pomodoroEndSound.play();
                    showStretchingModal(); 
                    showStretching();
                    playPauseIcon.classList.replace('fa-pause', 'fa-play');
                  }
                }, 1000);
              }
              isPomodoroRunning = true;
            }
          }
Explicação passo a passo:
- Verificação do Estado do Pomodoro:
A função primeiro verifica se o Pomodoro já está em execução para evitar a reinicialização do temporizador atual.
- Preparação do Temporizador:
Se o temporizador do Pomodoro não estiver ativo (countdownInterval não estiver definido), a função prepara o temporizador. Isso inclui definir a duração do Pomodoro (recuperada do localStorage ou valor padrão de 25 minutos) e reiniciar o timeLeftInSeconds.
- Início da Contagem Regressiva:
Utiliza setInterval para criar um temporizador que decrementa timeLeftInSeconds a cada segundo. A cada decremento, a função updateCountdownDisplay() é chamada para atualizar visualmente o tempo restante na interface do usuário.
- Conclusão do Pomodoro:
Quando o tempo chega a zero, a contagem regressiva é interrompida (clearInterval), o estado do Pomodoro é atualizado para não rodando (isPomodoroRunning = false), o som de término do Pomodoro é reproduzido, e o modal de alongamento é exibido através da função showStretchingModal(). Além disso, a função showStretching() é chamada para selecionar e preparar um alongamento específico.
- Atualização do Ícone de Controle:
O ícone no botão de controle é atualizado de "play" para "pause", refletindo que o Pomodoro está ativo.
pausePomodoro()
O QUE FAZ:
A função pausePomodoro() é projetada para pausar o temporizador do Pomodoro caso ele esteja ativo. Isso é feito interrompendo a contagem regressiva e atualizando o estado da aplicação para refletir que o Pomodoro não está mais rodando.
A FUNÇÃO:
function pausePomodoro() {
            if (isPomodoroRunning && countdownInterval) {
              clearInterval(countdownInterval);
              countdownInterval = null;
              isPomodoroRunning = false;
            }
          }
Explicação passo a passo:
- Verificação do Estado do Pomodoro:
Primeiro, a função verifica se o Pomodoro está atualmente em execução (isPomodoroRunning é true) e se existe um intervalo de contagem regressiva ativo (countdownInterval não é null). Esta etapa assegura que a função só tentará pausar o Pomodoro se ele estiver efetivamente rodando.
- Parada da Contagem Regressiva:
Utiliza clearInterval(countdownInterval) para parar o intervalo de contagem regressiva que decrementa timeLeftInSeconds a cada segundo. Esta ação interrompe efetivamente a contagem regressiva do Pomodoro.
- Reinicialização do Intervalo de Contagem Regressiva:
A variável countdownInterval é definida como null após a interrupção do intervalo. Isso indica que não há uma contagem regressiva em execução no momento e permite que a contagem regressiva seja reiniciada posteriormente sem conflitos.
- Atualização do Estado do Pomodoro:
O estado isPomodoroRunning é definido como false, refletindo que o Pomodoro foi pausado e não está mais ativo.
resetPomodoro()
O QUE FAZ:
A função resetPomodoro() é responsável por reiniciar completamente o temporizador do Pomodoro, configurando-o de volta para a duração inicial configurada pelo usuário ou para o valor padrão. Ela interrompe qualquer contagem regressiva em andamento, reseta o estado do Pomodoro para não ativo, e atualiza a interface do usuário para refletir essas mudanças.
A FUNÇÃO:
function resetPomodoro() {
            if (countdownInterval) {
              clearInterval(countdownInterval);
              countdownInterval = null;
            }
            isPomodoroRunning = false;

            const durationMinutes =
              parseInt(localStorage.getItem('pomodoroDuration')) || 25;
            timeLeftInSeconds = durationMinutes * 60;

            updateCountdownDisplay();

            const playPauseIcon = document.getElementById('playPauseIcon');
            if (playPauseIcon.classList.contains('fa-pause')) {
              playPauseIcon.classList.replace('fa-pause', 'fa-play');
            }
          }
Explicação passo a passo:
- Parada da Contagem Regressiva Ativa:
Se houver uma contagem regressiva em andamento (indicado pela existência de countdownInterval), a função a interrompe com clearInterval(countdownInterval) e define countdownInterval como null. Isso garante que a contagem regressiva anterior seja completamente interrompida antes de reiniciar.
- Reconfiguração do Estado do Pomodoro:
Define isPomodoroRunning como false, indicando que o Pomodoro não está mais rodando. Isso é necessário para garantir que o estado da aplicação reflita corretamente a ação de reset.
- Definição do Tempo Inicial:
Configura timeLeftInSeconds para o valor especificado nas configurações do usuário armazenadas em localStorage ou, na ausência deste, para o valor padrão de 25 minutos. Isso prepara o temporizador para uma nova sessão com a duração correta.
- Atualização da Interface do Usuário:
Chama a função updateCountdownDisplay() para atualizar a exibição do contador com o novo tempo de início. Isso garante que a interface do usuário mostre corretamente o tempo total disponível para a próxima sessão do Pomodoro.
- Ajuste do Ícone do Botão de Play/Pause:
Se o ícone do botão de play/pause estiver mostrando "pause" (indicado pela classe fa-pause), ele é substituído por "play" (indicado pela classe fa-play). Isso reflete a ação de reset na interface do usuário, mostrando que o Pomodoro está pronto para ser iniciado novamente.
updateCountdownDisplay()
O QUE FAZ:
A função updateCountdownDisplay() atualiza a exibição do contador na interface do usuário para refletir o tempo restante atual no temporizador do Pomodoro. Ela calcula os minutos e segundos a partir do valor total em segundos (timeLeftInSeconds) e formata esses valores para um formato legível de relógio (MM:SS), garantindo que sempre haja dois dígitos para minutos e segundos.
A FUNÇÃO:
            function updateCountdownDisplay() {
              const minutes = Math.floor(timeLeftInSeconds / 60);
              const seconds = timeLeftInSeconds % 60;
              document.getElementById('pomodoroCountdown').textContent = `${String(
                minutes
              ).padStart(2, '0')}:${String(seconds).padStart(2, '0')}`;
            }
Explicação passo a passo:
- Cálculo dos Minutos e Segundos:
A variável minutes é calculada dividindo timeLeftInSeconds por 60 e arredondando para baixo para o número inteiro mais próximo com Math.floor(). Isso garante que apenas a parte inteira da divisão seja considerada para os minutos.
A variável seconds é calculada usando o operador módulo (%) em timeLeftInSeconds, o que retorna o resto da divisão de timeLeftInSeconds por 60. Isso efetivamente extrai a quantidade de segundos que sobram após os minutos serem calculados.
- Atualização do Texto do Contador:
O método String().padStart(2, '0') é usado para converter tanto minutes quanto seconds em strings e garantir que cada um tenha pelo menos dois dígitos, adicionando um zero à esquerda se necessário. Isso é importante para manter o formato do tempo consistentemente legível, independentemente da quantidade de tempo restante.
O elemento HTML com o ID pomodoroCountdown é selecionado, e seu conteúdo de texto é atualizado com a string formatada que representa o tempo restante no formato MM:SS.

A função startStretchingPause() inicia a contagem regressiva para uma pausa no Pomodoro, que pode ser uma pausa curta ou longa dependendo do número de sessões de Pomodoro completadas. A duração da pausa é determinada pelas configurações salvas no localStorage, e a função atualiza dinamicamente a exibição do contador de pausa na interface do usuário.
A FUNÇÃO:
function startStretchingPause() {
            const pauseCount = parseInt(localStorage.getItem('pause')) || 0;
            const isLongPause = pauseCount === 4;
          
            const pauseDuration = isLongPause
              ? parseInt(localStorage.getItem('pomodoroLongPause'))
              : parseInt(localStorage.getItem('pomodoroPause'));
          
            pauseTimeLeftInSeconds = pauseDuration * 60;
            updatePauseCountDownDisplay();
          
            pauseCountDownInterval = setInterval(() => {
              if (pauseTimeLeftInSeconds > 0) {
                pauseTimeLeftInSeconds--;
                updatePauseCountDownDisplay();
              } else {

                pauseCountDownInterval = null;

Explicação passo a passo:
- Determinação do Tipo de Pausa:
A função primeiro verifica o número atual de pausas completadas (pauseCount), obtido do localStorage. Se pauseCount é igual a 4, indica que é hora de uma pausa longa.
- Cálculo da Duração da Pausa:
A duração da pausa (pauseDuration) é determinada com base em se a pausa atual deve ser longa ou curta, consultando as configurações salvas no localStorage para pomodoroLongPause ou pomodoroPause, respectivamente.
- Início da Contagem Regressiva da Pausa:
pauseTimeLeftInSeconds é configurado para o total de segundos da pausa escolhida, convertendo pauseDuration em minutos para segundos.
A função updatePauseCountDownDisplay() é chamada inicialmente para atualizar a interface do usuário com o tempo total da pausa.
- Contagem Regressiva Ativa:
Um intervalo de contagem regressiva (setInterval) é iniciado, decrementando pauseTimeLeftInSeconds a cada segundo e atualizando a interface do usuário chamando updatePauseCountDownDisplay() a cada decremento.
- Conclusão da Pausa:
Quando pauseTimeLeftInSeconds chega a 0, o intervalo é limpo (usando clearInterval), o que encerra a contagem regressiva da pausa.
O som de fim de pausa (pomodoroEndPauseSoung.play()) é reproduzido para notificar o usuário de que a pausa terminou.

A função incrementPauseCount() gerencia e atualiza o contador de pausas do Pomodoro armazenado no localStorage. O método incrementa o contador de pausas após cada sessão de Pomodoro, reiniciando o contador depois de atingir quatro pausas (indicando que a próxima pausa deverá ser uma pausa longa, conforme a metodologia Pomodoro).

O novo valor de pauseCount é convertido para uma string e salvo no localStorage, atualizando o registro do número de pausas completadas até o momento.

O QUE FAZ:
A função updateLevelProgress() atualiza o progresso do usuário em relação aos alongamentos dentro de um determinado nível de dificuldade. Calcula o total de alongamentos disponíveis para a dificuldade atual, subtrai os já completados, e atualiza a interface do usuário para refletir os alongamentos restantes. async function updateLevelProgress() {
            const usernameInput = document.getElementById('username'); 
            const username = usernameInput ? usernameInput.value : null; 
            const difficulty = localStorage.getItem('difficulty');
            let key = `previous${
              difficulty.charAt(0).toUpperCase() + difficulty.slice(1)
            }Stretchings`;

            const stretchings = await loadStretchings();

              );
            }
          }
