INSTRUÇÕES PARA USAR ESSA ESSA PÁGINA

No site API-NINJAS

  • Acesse o site API-NINJAS - https://api-ninjas.com
  • Caso não tenha se cadastrado clique em "Sign Up" ou clique aqui
  • Depois de se cadastrar e fazer o Login:
    • Clique em "My Account" ou aqui
    • Procure por API Key e clique no botão "Show API KEY"
    • Copie a API KEY vai ser algo nesse estilo: E04PvGN89wI1PdcdZNjZu/Qw==gBNl5v5l8vs0w2E04PvGN89w

No diretório principal deste repositório

  • Abra o arquivo config.json
  • Cole sua chave onde diz SUA_API_KEY_COPIADA_COLE_AQUI
  • Vai ficar assim: "API_KEY": "E04PvGN89wI1PdcdZNjZu/Qw==gBNl5v5l8vs0w2E04PvGN89w"

Criar o arquivo .gitignore

  • Dentro do .gitignore apenas escreva config.json
  • Salve
  • Pronto a partir de agora o git não vai enviar mais o config.json que tem sua chave api

Se esta tudo ok clique em "Ver Alongamentos" que a função getApiStreching() do arquivo get-stretching.js vai retornar todos alongamento da API
